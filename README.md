# Entrega 3 - SPA (Minha Conta)

Projeto criado para atender à entrega 3 da disciplina.

**Requisitos atendidos**:
- SPA (Single Page Application) com roteador hash
- Sistema de templates em `js/templates.js`
- Validação de formulário com mensagens de erro (`js/formValidation.js`)
- Armazenamento local via `localStorage`
- Código modular (pasta `js/`)
- Estrutura de pastas: `css/`, `js/`, `img/`, `index.html`

**Como usar**:
- Abra `index.html` localmente ou use GitHub Pages (o repositório pode ser publicado em https://<usuario>.github.io/entrega3/)

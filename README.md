# Entrega Final - Minha Conta (Entrega 4)

Projeto final que consolida SPA acessível, GitFlow e otimização para produção.

## O que foi implementado

- SPA com roteador hash
- Templates JS (js/templates.js)
- Validação de formulário e mensagens (js/formValidation.js)
- localStorage para armazenar usuários
- Acessibilidade: navegação por teclado, skip link, ARIA labels, foco visível
- Tema rosa com contraste ajustado + dark mode + high contrast
- Minified assets em css/min/ e js/min/
- README e arquivos de suporte para GitFlow

## GitFlow & Versionamento
Branches usadas:
- main (produção)
- develop (desenvolvimento)
- feature/acessibilidade
- feature/otimizacao
- docs/readme-profissional

Commits semânticos de exemplo:
- feat(accessibility): add keyboard navigation support
- fix(form-validation): improve screen reader messages
- chore(build): add minified assets

Release: `v1.0.0` (exemplo)

## Como rodar localmente
1. Abra `index.html` no navegador ou publique via GitHub Pages.

## WCAG 2.1 (AA) — checklist aplicado
- Skip link
- Focus visible and keyboard navigation
- ARIA roles and labels
- Contrast adjusted (>=4.5:1 for normal text)
- Dark mode and High contrast toggles

## Notas de produção
- Arquivos minificados em `css/min` e `js/min`.
